| 编号        | 功能名称                 | 功能描述                                         |
| --------- | -------------------- | -------------------------------------------- |
| FR-DR-001 | 入园申请提交               | 司机在入园前提交入园申请（访问时间、货物、车辆、司机信息），用于园区提前审核与安排。   |
| FR-DR-002 | 审核结果查询/接收            | 司机查询/接收入园申请的审核结果（通过/拒绝/待审核），据此决定是否按计划到访。     |
| FR-DR-003 | 到园扫码签到（不放行）          | 司机到达园区门口扫描二维码完成签到与计划核验，仅登记到访，进入排队流程，不直接放行入园。 |
| FR-DR-4   | 排队登记与叫号通知（含月台分配）     | 系统为已签到司机分配排队号，按规则叫号并通知其前往指定月台/通道。            |
| FR-DR-005 | 叫号入园放行（闸机联动）         | 司机被叫号后，系统联动门禁/道闸放行入园，引导车辆驶往指定月台。             |
| FR-DR-006 | 月台到达确认（车牌识别/人工录入）    | 车辆到达月台后由摄像头自动识别车牌或月台管理员录入车牌，系统将该车辆状态置为“作业中”。 |
| FR-DR-007 | 司机作业申报提交             | 卸货/装货完成后，司机在系统中提交本次作业申报（实际货物/数量等），进入核验流程。    |
| FR-DR-008 | 货物清点与核验关联（后台）        | 月台管理员清点货物，选择对应司机的作业申报记录进行核验并添加核验记录。          |
| FR-DR-009 | 核验记录生成与流转（后台）        | 系统生成核验记录并流转给客户管理员进行确认，支持差异标注。                |
| FR-DR-010 | 客户管理员核验确认/反馈（后台）     | 客户管理员对核验记录确认无误或提出差异处理意见，形成最终核验结果。            |
| FR-DR-011 | 后续指令通知（离园/等待/前往下一月台） | 根据核验结果与调度策略，系统向司机下发后续指令：离园、继续等待，或前往下一月台。     |
| FR-DR-012 | 离园放行（出口联动）           | 司机仅在收到“离园通知”后方可离园；系统联动出口道闸放行并形成出园记录。         |
